# protonmail-ui-optimizer
Removes promotional banners and offer buttons from a Proton Mail free account for a cleaner, distraction-free experience.

## Features
- Automatically hides promo/offer buttons site-wide
- Removes promotional banners dynamically

## Installation
1. Install the Tampermonkey browser extension or your preferred browser userscript manager.
2. Copy the script from protonmail-ui-optimizer.js into a new userscript.
3. Save and reload Proton Mail or account pages.
